Restaurant Extension

This module depends on point of sale. Using separately from restaraunt odoo base module

Main Features:

  • Adding new field Transfer under Point of Sale Selling Screen. Use the field to transfer/ merge/ split table.
  • If the destination table already has an order, the system will create a new order.

  • Customise current Order function in order to allow to connect directly to printer without configure the proxy to be able to print in Bar/ Kitchen.
